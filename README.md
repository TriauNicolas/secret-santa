# Secret-santa

Exercice fait dans le cadre d'un test technique pour l'entreprise UNiFAi !

Dépendances installées :
- Vue
- Buefy
- JSON-Server
- Faker
- npm-run-all

## Project setup
```
npm install
```

### Run API with fake data & run project
```
npm run all
```

### Just generate data on fakeAPI
```
npm run generate
```
